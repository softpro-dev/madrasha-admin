## Project Overview

**madrasha-admin** is a Nuxt 4 + Vue 3 admin dashboard for Madrasha (Islamic school) management system. Currently in early stages with minimal setup.

## TypeScript Requirement

**All code must use TypeScript.** No plain JavaScript files. Use `.ts` files and `<script lang="ts">` in Vue components. See CLAUDE.local.md for details.

## Project Structure

- `app/` — Vue components and pages (Nuxt auto-discovers `app/pages/`, `app/components/`, `app/layouts/`)
- `public/` — Static assets (images, favicons, etc.)
- `nuxt.config.ts` — Nuxt configuration
- `tsconfig.json` — References generated Nuxt TypeScript configs in `.nuxt/`
- `.nuxt/` — Generated by `nuxt prepare` (git-ignored)

## Nuxt Conventions

This project uses Nuxt 4's file-based routing:
- Pages go in `app/pages/` (auto-routed by file path)
- Reusable components go in `app/components/` (auto-imported)
- Layouts in `app/layouts/` (wrap page content)

Example structure for adding a feature:
```
app/
  pages/
    students/
      index.vue        → /students
      [id].vue         → /students/:id
  components/
    StudentCard.vue
    StudentForm.vue
```
